Introduction

Projet de Présentation de Données Web


Angular, NodeJs, 3DJS, Jquery, Boostrap, MongoDb, Json, Rest.


Sart Apache:

  • manager-osx

Sart Node Js:
  • cd /usr/local/lib/node_modules
  • sudo node nodejsserver.js

Sart Mongo Db:
  • Server: /usr/local/bin/mongodb/bin/mongod
  • Terminal: /usr/local/bin/mongodb/bin/mongod --dbpath /data/db
  • Data Path: /data/db
